# Voyage Images

This directory should contain images for the travel quiz.

## Image Naming Convention

Images should be named according to the voyage ID from your database:

- `1.jpg` - for voyage with idVoyage = 1 (Addis Ababa)
- `2.jpg` - for voyage with idVoyage = 2 (Shanghai)
- `3.jpg` - for voyage with idVoyage = 3 (Berlin)
- `4.jpg` - for voyage with idVoyage = 4 (Toronto)
- `5.jpg` - for voyage with idVoyage = 5 (Algiers)
- `6.jpg` - for voyage with idVoyage = 6 (Rio de Janeiro)
- `7.jpg` - for voyage with idVoyage = 7 (Casablanca)
- `8.jpg` - for voyage with idVoyage = 8 (Algiers)

## Image Requirements

- Format: JPG or JPEG
- Recommended size: 800x600 pixels or larger
- Content: Clear images of the travel destinations
- File size: Keep under 2MB for optimal loading
